cims
====

Command line toolbelt for NYU's cims machines based on Heroku Toolbelt

##Login

```bash
$ cims login
```

##Run command

```bash
$ cims run "ls -al"
```

##Upload File
```bash
$ cims upload <path-to-file>
```

##Download File
```bash
$ cims download <path-to-file>
```

##Use cims version of matlab
```bash
$ cims matlab
```
